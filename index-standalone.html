npm run prod

Step 3: Open in Browser

Docker Deployment

Quick Start:
docker-compose up Your app will be available at: http://localhost:5000

For any server:
1. Install Docker & Docker Compose
2. Clone your repository
3. Run docker-compose up -d
4. Access on your-server-ip:5000

×

Heroku Deployment (Free)

Install Heroku CLI:
npm install -g heroku

Deploy:
heroku login
heroku create your-app-name
git push heroku main
heroku open

×

AWS Deployment (Elastic Beanstalk)

Install AWS CLI & EB CLI:
pip install awscli awsebcli

Deploy:
aws configure
eb init -p "Node.js 18 running on 64bit Amazon Linux 2"
eb create quiz-interactive
eb deploy
eb open
